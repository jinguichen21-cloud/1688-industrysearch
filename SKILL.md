# 1688工业品搜索技能

## 核心能力

在用户意图涉及寻找工业品商品或工业品批量采购时触发，提供从 1688 搜索工业品相关商品的能力。

### 适用场景

- **工业品货源搜索** - 找螺栓、螺母、铆钉、包装材料、电子元器件等工业原材料
- **批量采购比价** - 对比多个供应商的价格、销量、服务承诺
- **以图搜货** - 根据商品图片快速找到相似货源
- **行家推荐筛选** - 优先展示"行家力荐"商品，降低采购决策成本

### 不适用场景

- C 端零售商品搜索（请使用其他 1688 选品技能）
- 下单付款、物流查询、库存管理、售后处理
- 非工业品类的商品搜索

## 搜索模式

仅支持以下两种搜索模式，**不支持图文混合搜索**：

| 模式 | 适用场景 | 入参 | 脚本 |
|------|---------|------|------|
| 文本搜索 | 用户用文字描述需求 | `query` | `scripts/text_search.py` |
| 图片搜索 | 用户提供商品图片 | `imageUrl` | `scripts/image_search.py` |

**重要规则**：如果用户同时提供了图片和文字描述，**仅保留图片要求**，构造 `imageUrl` 进行图片搜索。

## 执行流程

### Step 1: 判断搜索模式并准备参数

#### 文本搜索场景

根据需求复杂度将用户描述拆解为短词，用空格分隔。**每次搜索建议不超过 3 个短词**。

改写示例：

| 用户原始描述 | 改写后的 query |
|------------|---------------|
| 帮我找 m6 不锈钢螺栓 | `m6 不锈钢 螺栓` |
| 我想批量采购 304 不锈钢外六角螺丝 | `304 不锈钢 外六角 螺丝` |
| 有没有工业用的碱性电池 | `工业 碱性 电池` |

#### 图片搜索场景

直接使用用户提供的图片 URL 作为 `imageUrl`。

#### 默认参数

- 如用户无特殊要求，每次默认返回 **10** 个商品（`itemCount: 10`）

### Step 2: 调用搜索接口

根据搜索模式选择对应的脚本执行：

#### 文本搜索

```bash
python scripts/text_search.py "m6 不锈钢 螺栓" 10
```

#### 图片搜索

```bash
python scripts/image_search.py "https://img.alicdn.com/imgextra/i1/xxx.png" 10
```

如需更大输出量，可调整第二个参数（最大建议值：50）。

### Step 3: 格式化输出结果

脚本会自动格式化 API 返回的商品列表，包含以下关键信息：

1. **商品标题** - 带详情页链接
2. **价格** - 单价（元为单位）
3. **销量** - 如有销量数据
4. **店铺名称**
5. **行家推荐** - 如有，突出展示 🏆 标识
6. **服务保障** - 选取关键服务标签（晚发必赔、48小时发货等）
7. **商品属性** - 如有 cpvTagList，展示属性标签

### Step 4: 异常处理

| 场景 | 处理方式 |
|------|---------|
| 接口返回 `fail: true` 或 `success: false` | 向用户提示搜索失败，展示 `errorMsg` 内容（如有），建议用户换个关键词重试 |
| 返回 `data` 为空数组 | 提示未找到相关商品，建议用户调整搜索关键词或换用图片搜索 |
| 图片 URL 无效或不可访问 | 提示图片地址可能有误，建议用户重新提供图片链接 |
| 网络超时或连接失败 | 提示网络问题，建议检查网络连接后重试 |

## 接口参考

详细 API 文档请参见 `references/api_reference.md`。

### 接口地址

| 搜索类型 | 接口地址 |
|---------|---------|
| 文本搜索 | `https://ainext.1688.com/1688claw/industrySkill/textSearch` |
| 图片搜索 | `https://ainext.1688.com/1688claw/industrySkill/imageSearch` |

### 请求参数

```json
{
  "query": "m6 不锈钢 螺栓",  // 文本搜索专用
  "imageUrl": "https://...",  // 图片搜索专用
  "itemCount": 10
}
```

### 响应字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `itemId` | number | 商品 ID |
| `itemTitle` | string | 商品标题 |
| `expertRecommend` | string/null | 行家推荐标签 |
| `itemDetailUrl` | string | 商品详情页链接 |
| `sellerName` | string | 店铺名称 |
| `itemSaleCount` | string/null | 销量描述 |
| `serviceList` | string[] | 服务标签列表 |
| `itemPrice` | string | 商品单价（元） |
| `cpvTagList` | string[]/null | 商品属性标签 |

## 使用说明

### 触发条件

当用户表达以下意图时自动触发此技能：

- "帮我找 XX 工业品"
- "采购一批 XX"
- "有没有卖 XX 的厂家"
- "XX 多少钱批发"
- "找 XX 货源"
- "以图搜 XX"

### 使用示例

**文本搜索示例：**

```
用户：帮我找 m6 不锈钢螺栓，要包邮的
→ 执行：python scripts/text_search.py "m6 不锈钢 螺栓" 10
```

**图片搜索示例：**

```
用户：这张图里的螺丝哪里有卖？[图片 URL]
→ 执行：python scripts/image_search.py "https://..." 10
```

**混合输入处理：**

```
用户：找这种螺栓 [图片]，要 304 不锈钢材质的
→ 仅使用图片搜索，忽略文字描述中的材质要求
→ 执行：python scripts/image_search.py "https://..." 10
```

## 兼容性说明

- **运行环境**：Python 3.6+
- **网络要求**：需能访问 1688 预发环境接口
- **依赖库**：仅需 Python 标准库（urllib、json）

## 注意事项

1. **搜索词优化**：避免过长句子，拆分为 2-3 个核心关键词效果更好
2. **图片质量**：图片搜索需要清晰的商品正面图，模糊图片可能影响匹配精度
3. **结果时效性**：商品价格和库存可能实时变化，详情页信息为准
4. **采购决策**：本技能仅提供信息检索，实际采购请通过 1688 官方渠道核实
